# Seanime i18n

This document describes the localization system used by this fork.

This repository started as an i18n / Traditional Chinese fork, but the fork now also carries Docker, updater, playback, and release workflow changes. For those broader fork-specific changes, see the main README and release notes.

## Locale files

Frontend locale files live in:

`seanime-web/src/i18n/locales/`

Current files:

- `seanime-web/src/i18n/locales/en-US.json`
- `seanime-web/src/i18n/locales/zh-TW.json`

The i18n bootstrap, shared labels, and language preference exports live in:

- `seanime-web/src/i18n/index.ts`
- `seanime-web/src/i18n/labels.ts`
- `seanime-web/src/i18n/provider.tsx`

## How translation works

- `en-US` is the default language.
- `en-US` is also the fallback language.
- `zh-TW` can be selected from the frontend settings page.
- Missing `zh-TW` keys fall back to `en-US`.
- The current language preference is stored locally on the client.

## Adding a translation key

1. Add the new key to `seanime-web/src/i18n/locales/en-US.json`.
2. Add the same key to `seanime-web/src/i18n/locales/zh-TW.json`.
3. Use the key from React with `useTranslation()` and `t("your.path")`.
4. If the string is shown from a non-React module, import the shared i18n instance from `@/i18n` and call `i18n.t("your.path")`.

## Adding another language

1. Create a new locale file in `seanime-web/src/i18n/locales/`, for example `ja-JP.json`.
2. Copy the full key structure from `en-US.json`.
3. Add the locale to `SUPPORTED_LANGUAGES` in `seanime-web/src/i18n/index.ts`.
4. Add the locale to the `resources` object in `seanime-web/src/i18n/index.ts`.
5. Add a visible label for the new language in the locale files.
6. Verify that missing keys still fall back to `en-US`.

## AniList display labels

- AniList genre and tag localization in this fork must stay display-only.
- Do not modify AniList raw data, filter values, query values, or cache values just to translate labels.
- Unknown genre or tag values should fall back to the original AniList string.
- Shared display mappings should live in `seanime-web/src/i18n/labels.ts`.
- Only reviewed AniList tag translations should be added to the runtime mapping.
